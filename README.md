# Smart-File-Organizer
A Python-based automation tool that organizes files into categories like Images, Documents, and Videos based on file type. It helps users clean and manage messy folders efficiently with minimal manual effort.

## 📌 Description
A Python script that automatically organizes files into folders like Images, Documents, and Videos based on file type.

## 🚀 Features
- Detects file types  
- Creates folders automatically  
- Moves files accordingly  

## ▶️ How to Run
python organizer.py
